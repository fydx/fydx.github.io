Socket相关

Socket是网络交互主要使用的一种方式，我们用一个例子来说明。

这是一个QQ客户端，分为服务器端和客户端

服务器端代码如下

package chat;  

import java.io.BufferedReader;  
import java.io.IOException;  
import java.io.InputStreamReader;  
import java.io.PrintWriter;  
import java.net.ServerSocket;  
import java.net.Socket;  
import java.util.ArrayList;  
import java.util.List;  
import java.util.concurrent.ExecutorService;  
import java.util.concurrent.Executors;  

public class chat {  
    private static final int PORT = 8888; // 端口号  
    private static List<Socket> list = new ArrayList<Socket>(); // 保存连接对象
    private ExecutorService exec;
    private ServerSocket server;
public static void main(String[] args) {  
    new chat();  
}  

public chat() {  
    try {  
        server = new ServerSocket(PORT);  
        exec = Executors.newCachedThreadPool();  
        System.out.println("服务器已启动！");  

        Socket client = null;  
        while (true) {  
            client = server.accept(); // 接收客户连接  
            list.add(client);  
            exec.execute(new ChatTask(client));  
        }  
    } catch (IOException e) {  
        e.printStackTrace();  
    }  
}  

static class ChatTask implements Runnable {  
    private Socket socket;  
    private BufferedReader br;  
    private PrintWriter pw;  
    private String msg;  

    public ChatTask(Socket socket) throws IOException {  
        this.socket = socket;  
        br = new BufferedReader(new InputStreamReader(socket  
                .getInputStream()));  
        msg = "【" + this.socket.getInetAddress() + "】进入聊天室！当前聊天室有【"  
                + list.size() + "】人";  

        sendMessage();  
    }  

    public void run() {  
        try {  
            while ((msg = br.readLine()) != null) {  
                if (msg.trim().equals("bye")) {  
                    list.remove(socket);  
                    br.close();  
                    pw.close();  
                    msg = "【" + socket.getInetAddress() + "】离开聊天室！当前聊天室有【"  
                            + list.size() + "】人";  
                    socket.close();  

                    sendMessage();  
                    break;  
                } else {  
                    msg = "【" + socket.getInetAddress() + "】说：" + msg;  
                    sendMessage();  
                }  
            }  
        } catch (IOException e) {  
            e.printStackTrace();  
        }  
    }  

    /** 
     * 群发消息给聊天室的所有人 
     */  
    private void sendMessage() throws IOException {  
        System.out.println(msg);  
        for (Socket client : list) {  
            pw = new PrintWriter(client.getOutputStream(), true);  
            pw.println(msg);  
        }  
    }  
}  
}

客户端代码如下：

package client;

import java.io.BufferedReader;  
import java.io.InputStreamReader;  
import java.io.PrintWriter;  
import java.net.Socket;  
import java.util.concurrent.ExecutorService;  
import java.util.concurrent.Executors;  

public class client {  
private static final int PORT = 8888;  
private static ExecutorService exec = Executors.newCachedThreadPool();  

public static void main(String[] args) throws Exception {  
    new client();  
}  

public client() {  
    try {  
        Socket socket = new Socket("192.168.1.100", PORT);
        exec.execute(new Sender(socket));
        System.out
                .println("【" + socket.getInetAddress() + "】您好，欢迎来到聊天室！");

        BufferedReader br = new BufferedReader(new InputStreamReader(socket
                .getInputStream()));
        String msg;
            while ((msg = br.readLine()) != null) {
                System.out.println(msg);
            }
    } catch (Exception e) {  

    }  
}  

/** 
 * 客户端线程获取控制台输入消息 
 */  
static class Sender implements Runnable {  
    private Socket socket;  

    public Sender(Socket socket) {  
        this.socket = socket;  
    }  

    public void run() {  
        try {  
            BufferedReader br = new BufferedReader(new InputStreamReader(  
                    System.in));  
            PrintWriter pw = new PrintWriter(socket.getOutputStream(), true);  
            String msg;  

            while (true) {  
                msg = br.readLine();  
                pw.println(msg);  

                if (msg.trim().equals("bye")) {  
                    pw.close();  
                    br.close();  
                    exec.shutdownNow();  
                    break;  
                }  
            }  
        } catch (Exception e) {  
            e.printStackTrace();  
        }  
    }  
}  
}

我们首先从服务器端说起

  1. 我们首先遇到的是ServerSocket这个类，ServerSocket负责建立对某个端口，我们以8888作为我们开启的端口号

    int PORT = 8888；
server = new ServerSocket(PORT);

  2. 接下来是一个线程池,用来接收用户的连接
    cachedThreadPool 是一个缓冲线程池，可以重用以前构造的线程，如果没有可用的就重新构建一个，缓冲线程池会终止并从池里面移除60秒未被使用的线程

    ExecutorService exec = Executors.newCachedThreadPool()

  3. 再往下，调用的是一个ServerSocket的accept方法，返回的是一个Socket对象，我们把这个Sokcet加入到一个List中，并且开启一个ChatTask线程以处理相关事务
    关于Socket的发送我们会在客户端的相关再详述

  4. 关于ChatTask，输出信息和读取Socket的流数据都会在这里进行

    br = new BufferedReader(new InputStreamReader(socket
             .getInputStream()));

    上面这个代码把Socket的输入字节流封装成InputStreamReader，编程字符输入流，最终用BufferedReader来从字符输入流中读取相关字符。

    msg = br.readLine()

    这样子即可最终拿到客户端发送来的内容

  5. 向所有客户端发送消息

    这里我们来读取到已经连接到服务端的所有客户端的Socket，获取PrintWriter向所有的客户端打印（输出）消息

    private void sendMessage() throws IOException {  
     for (Socket client : list) {  
         pw = new PrintWriter(client.getOutputStream(), true);  
         pw.println(msg);  
     }  
 }

接下来是客户端相关

客户端这里相对简单一些：

  1. 首先我们要明确服务端的ip和端口，这里以192.168.1.100为ip，8888为开启的端口

  2. 我们新建一个Socket对象

    Socket socket = new Socket("192.168.1.100", 8888);

  3. 我们还是可以去使用一个线程池，去获取控制台的相关信息

    ExecutorService exec = Executors.newCachedThreadPool()

  4. 客户端需要把自己输入的东西显示在自己的控制台上，这里基本上和服务端做的事情一样，读取Socket里面的内容，只不过这次读取的是客户端自己的Socket，而不是接收的

            .getInputStream()));

 PrintWriter pw = new PrintWriter(socket.getOutputStream(), true);

对于网络交互，最重要的是理解Socket的相关传输机制，发送，打印，输入流输出流。我们可以简单的理解为开辟一个通道，打开通道大门，双方通过一个包裹（Socket）交互，需要里面的信息，就把包裹拆了，读取其中的信息，不需要的时候，就把大门关闭。
