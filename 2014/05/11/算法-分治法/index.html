算法-分治法在归并排序和快速排序中的体现

分治法，就是把一个大的问题分解成若干个子问题，子问题可以进一步分解为更小的子问题，直到问题分解为一个可以直接求解的子问题，然后把子问题的解合并即可得到大问题的解，分而治之，各个击破。基本上分治法都得基于递归去解决。

分治法保持分、治、合三个步骤求解。

分治法在归并排序和快速排序得到具体的体现，我们来研究一下

归并排序

以下代码引自维基百科-归并排序

public static int[] mergeSort(int[] arr){//归并排序 --递归
if(arr.length==1){
    return arr;
}
int half = arr.length/2;
int[] arr1 = new int[half];
int[] arr2 = new int[arr.length-half];
System.arraycopy(arr, 0, arr1, 0, arr1.length);
System.arraycopy(arr, half, arr2, 0, arr2.length);
arr1 = mergeSort(arr1);
arr2 = mergeSort(arr2);
return mergeSortSub(arr1,arr2);
}

private static int[] mergeSortSub(int[] arr1,int[] arr2){//归并排序子程序
int[] result = new int[arr1.length+arr2.length];
int i = 0;
int j = 0;
int k = 0;
while(true){
    if(arr1[i] < arr2[j]){
        result[k] = arr1[i];
        if(++i>arr1.length-1){
            break;
        }
    }else{
        result[k] = arr2[j];
        if(++j>arr2.length-1){
            break;
        }
    }
    k++;
}
for(;i<arr1.length;i++){
    result[++k] = arr1[i];
}
for(;j<arr2.length;j++){
    result[++k] = arr2[j];
}
return result;
}

我们来具体看看是如何“分治“是如何在归并排序里面体现的

  1. 首先我们要理解归并排序的做法：简单的说，就是把两个已经排序完毕的数组合并成一个有序数组。 形象一点说，就是把两摞已经排好顺序的扑克牌合并成一摞。 假设这两摞牌都是升序排列（也就是说最上面的牌是最小的），那么，做这个排序的事情的时候，我们比较两摞牌最上面的牌的大小，把比较小的放在另外一摞的第一个，然后继续做比较，把小的放在新的牌堆底，直到某一摞牌都被放完了，空了，我们把另外一摞所有的牌都放在新的牌堆的底部。
  2. 在算法上，首先我们把一个数组平分为两部分，我们称之为arr1和arr2，我们开始进行递归，将arr1和arr2无限的分解，直到他们只被分解的剩下了1个元素，而每次递归的子程序都会执行归并排序
  3. 我们设定一个归并排序的子程序，传入两个数组，进行归并排序。我们开辟一个数组int[] result = new int[arr1.length+arr2.length];，长度是传入的两个数组长度之和。我们开启三个变量，i，j，k。 i的意义是传入第一个数组遍历到的位置，j为传入的第二个数组遍历到的位置，k是我们开辟的新数组指向的位置。接下来的操作就跟我们开始的时候说的“扑克牌”的操作类似，把传入两个数组大的那个数放到我们新开辟的数组里面，直到某个数组里面没有元素了（i或者j等于所指向的数组的长度），跳出循环，将没遍历完的数组的所有元素加入到新开辟的数组中，把新开辟的数组返回，归并排序子程序结束。
  4. 到现在，整个流程就比较清晰了，分解为直到数组里面剩了1个元素，从这里开始进行排序（1个数字肯定是有序的），接下来是两个数字的归并排序，3个数字的归并排序，以此类推，最终执行完整个数组的归并排序。

我们来看一下分、治、和在归并排序里面的具体体现：

  • 分 —— 分在归并排序里面就是无限把一个数组分成两个数组，直到分成的新数组里面剩了1个元素
  • 治 —— 主要是指归并排序子程序中对比两个数组i/j所指的大小，并加入到新数组中
  • 合 —— 在归并排序子程序中两个有序数组合并成一个有序数组，并返回

快速排序

相同的道理，快速排序也是类似的实现方式

public static void quicksort(int n[], int left, int right) {
    int dp;
    if (left < right) {
        dp = partition(n, left, right);
        quicksort(n, left, dp - 1);
        quicksort(n, dp + 1, right);
    }
}

public static int partition(int n[], int left, int right) {
    int pivot = n[left];
    while (left < right) {
        while (left < right && n[right] >= pivot)
            right--;
        if (left < right)
            n[left++] = n[right];
        while (left < right && n[left] <= pivot)
            left++;
        if (left < right)
            n[right--] = n[left];
    }
    n[left] = pivot;
    return left;
}

快速排序的步骤可以这样子理解：

  1. 在数组里面挑选出来一个元素，作为基准（pivot），这里挑这个元素，这里的可以选择挑选第一个。不过，最好是随机的选一个，做一个随机化快速排序，因为挑选的基准会影响到排序的效率，如果说挑选的是一个元素恰好是有序数组的第一个或者最后一个，会造成快速排序的最坏情况（O(n^2)）。
  2. 把数组里面比基准值大的元素放在后边，比基准值小的放在前面，相同的放在任意一边。
  3. 把基准值左边和右边的数组再当成一个新的数组，从第一步开始执行，直到递归到最后数组的大小是0或者1。

最后，我们来看一下在快速排序里面分治法的体现

  • 分 —— 基准元素左右两边各作为一个子数组
  • 治 —— 体现在partition中，分区，返回一个基准值所对应的位置
  • 合 —— 所有的操作实际上都是在原始数组中做的操作，直接会返回这个排序好的数组
